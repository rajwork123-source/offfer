Step 1 of 6
Download & Install
MetaTrader 5
MetaTrader 5 is free. It includes the trading platform AND MetaEditor (the code editor) in one installer.
A
Download the installer
From MetaQuotes' official site
1
Open your browser and go to metatrader5.com
2
Click the large "Download MetaTrader 5" button on the homepage. This downloads a file called mt5setup.exe (Windows) or the equivalent for Mac/Linux.
3
Run the installer. Click Next → Next → Install. Accept all defaults — don't change the install folder unless you have a reason to.

Mac users: MT5 has a native Mac app available in the App Store for free. Search "MetaTrader 5" — it's the official one from MetaQuotes Software Corp.

Step 2 of 6
Create a Free
Demo Account
A demo account gives you virtual money ($10,000 by default) to test your EAs with real market data. It costs nothing and requires no personal financial information.
A
Open account dialog
If you closed it after install
1
If the "Open an Account" window closed, go to the top menu: File → Open an Account
File›Open an Account
B
Choose a broker & create demo
Use MetaQuotes-Demo for practice — no broker required
1
In the search box, type MetaQuotes and select MetaQuotes-Demo from the list. This is a virtual broker provided by MetaQuotes purely for practice.
2
Click Next. Select "Open a demo account to practice trading".
3
Fill in any name and email (can be fake for demo). Set Account currency to USD, Deposit to $10,000, and Leverage to 1:100.
4
Click Next. MT5 will create the account and show your Login number and Password. Write these down — you'll need them if MT5 ever asks you to log in again.
5
Click Finish. The platform will connect and you'll see a chart open with price data loading.
✅
Success sign: You'll see green "Connected" text in the bottom right corner of MT5, and a candlestick chart will appear.

Step 4 of 6
Write Your
First Expert Advisor
You'll create a simple EA that prints a message every time the price moves. It won't trade — it just proves your setup is working.
A
Create a new EA file
Using the MQL5 Wizard
1
In MetaEditor, go to File → New (or press Ctrl+N).
File›New›Expert Advisor (template)
2
The MQL5 Wizard opens. Select "Expert Advisor (template)" and click Next.
3
Set the Name to MyFirstEA. Leave all other fields at their defaults. Click Next → Finish.
4
MetaEditor generates a template file with OnInit(), OnTick(), and OnDeinit() functions already set up. This is your starting point.
B
Replace the code
Clear the template and paste this
1
Select all the code in the editor (Ctrl+A) and delete it.
2
Type or paste the following code exactly:
MQL5 — MyFirstEA.mq5
#property copyright "My First EA" #property version "1.00" #property strict // This runs ONCE when the EA starts int OnInit() { Print("✅ MyFirstEA has started!"); return(INIT_SUCCEEDED); } // This runs on EVERY price tick (price movement) void OnTick() { double bid = SymbolInfoDouble(_Symbol, SYMBOL_BID); Print("Current price: ", bid); } // This runs ONCE when the EA stops void OnDeinit(const int reason) { Print("👋 MyFirstEA has stopped."); }
💡
What this does: When started, it prints a confirmation. On every price movement, it reads and prints the current Bid price. When stopped, it says goodbye. No trades are placed — it's safe to run on any account.
Step 5 of 6
Compile the Code
& Attach to a Chart
Compiling converts your human-readable MQL5 code into a binary file (.ex5) that MT5 can execute. Then you drag it onto a chart to run it.
A
Compile the EA
Check for errors
1
Press F7 to compile, or click the Compile button (green play-like icon) in the toolbar.
2
Look at the bottom Toolbox panel. You should see:
Toolbox — Errors tab
MyFirstEA.mq5(1) : information: Compiling 'MyFirstEA.mq5'
MyFirstEA.mq5 : 0 error(s), 0 warning(s), compile time: 0:00.03
3
0 errors = success. A file named MyFirstEA.ex5 is now saved in MT5's Experts folder automatically.
⚠️
If you see errors: The Toolbox will show red text with a line number (e.g., MyFirstEA.mq5(12)). Double-click the error line to jump to the problem in your code. Most errors are typos — check for missing semicolons or misspelled function names.
B
Attach the EA to a chart
Back in the MT5 trading platform
1
Switch back to MT5 (click its icon in the taskbar, or press Alt+Tab).
2
In the left panel, click Navigator (or press Ctrl+N). Expand Expert Advisors. You should see MyFirstEA listed.
3
Double-click MyFirstEA in the Navigator. A settings dialog appears.
4
Make sure "Allow Algo Trading" is checked. Click OK.
5
The EA is now attached! You'll see a small smiley face icon ☺ in the top-right corner of the chart. A smiley face means the EA is running. A sad face means algo trading is disabled.
✅
Make sure "Algo Trading" is enabled globally. Check the MT5 toolbar for an "Algo Trading" button — it should be highlighted/active (green). If it's grey, click it to enable it.
Step 6 of 6
Read the Log —
Verify It's Working
The Experts tab is your window into what your EA is doing. Every Print() statement in your code shows up here. This is your #1 debugging tool.
A
Open the Experts tab
Where your Print() output appears
1
At the bottom of MT5, find the Terminal panel. If it's hidden, press Ctrl+T.
2
Click the Experts tab inside the Terminal panel.
3
You should see output like this:
Terminal — Experts tab
2024.01.15 09:00:01 MyFirstEA EURUSD,M1: ✅ MyFirstEA has started!
2024.01.15 09:00:01 MyFirstEA EURUSD,M1: Current price: 1.08542
2024.01.15 09:00:02 MyFirstEA EURUSD,M1: Current price: 1.08541
2024.01.15 09:00:03 MyFirstEA EURUSD,M1: Current price: 1.08543
B
Understanding the log format
Read every line like a pro
1
Timestamp — exact date and time the message was logged. Essential for tracking when something happened.
2
EA name + Symbol + Timeframe — tells you which EA on which chart generated the message (e.g., MyFirstEA EURUSD,M1).
3
Message text — whatever you passed to Print() in your code.
💡
Also check the "Journal" tab — this shows system-level messages from MT5 itself (connection status, order confirmations, errors from the broker). Experts tab = your code. Journal tab = MT5 system.
C
Stop the EA
Cleanly remove it from the chart
1
Right-click on the chart where the EA is running.
2
Select Expert Advisors → Remove. The smiley icon disappears.
3
In the Experts tab you'll see the OnDeinit() message: "👋 MyFirstEA has stopped."
